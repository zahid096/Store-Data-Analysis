# Store-Data-Analysis
Excel-based sales analysis &amp; interactive dashboard for Vrinda Store.

## Dataset
- File: '<a href="https://github.com/zahid096/Store-Data-Analysis/blob/main/Vrinda%20Store%20Data%20Analysis_using%20excel.xlsx">Vrinda Store Data</a>'
- Contents: sales transactions with columns like Date, Product, Category, Quantity, Price, Region, Customer

## Tools
- Microsoft Excel (Pivot Tables, Charts, Conditional Formatting)
- Exported screenshots  of dashboard in 'Dashborad of store data analysis'

## What I did
- Cleaned and prepared the data in Excel (removed duplicates, fixed dates).
- Performed Exploratory Data Analysis using Pivot Tables and Charts.
- Built an interactive dashboard (slicers, pivot charts) to visualize:
  - Top-selling products and categories
  - Monthly and seasonal sales trends
  - Regional sales comparison
